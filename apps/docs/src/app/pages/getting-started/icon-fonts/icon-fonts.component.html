Icon Fonts

Anglify comes bootstrapped with support for Material Design Icons, Material Icons, Font Awesome 4 and Font Awesome 5. By default, applications will default to use Material Design Icons.

Usage

To change your font library you have to provide the IconSettings token.

Afterwards the respective icon font must be imported. In the next section the installation process of each icon font is described separately.

Installing icon fonts

You are required to include the specified icon library (even if using default). This can be done by including a CDN link or importing the icon library into your application.

When changing icon fonts, make sure that you remove the old imports, as this can improve the performance of the website.
