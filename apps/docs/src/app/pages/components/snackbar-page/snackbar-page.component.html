Snackbar

Snackbars inform users of a process that an app has performed or will perform. They appear temporarily, towards the bottom of the screen. They shouldn't interrupt the user experience, and they don't require user input to disappear.

Snackbars contain a single line of text directly related to the operation performed. They may contain a text action, but no icons. You can use them to display notifications.

Examples

Basic

If you use the simple open() method, receiving a return or close value is impossible. Instead, you can use the open$() method, which returns an observable you can subscribe to.

Indefinitely

If you set the timeout property to 0, the snackbar will stay open forever.
